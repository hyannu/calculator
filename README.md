# 🧮 Modern Calculator

A modern and responsive calculator built using **HTML, CSS, and JavaScript**. 
It performs basic arithmetic operations with a clean glassmorphism-inspired user interface and keyboard support.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Percentage Calculation
- 🗑️ Clear Button
- ⌫ Backspace Button
- ⌨️ Keyboard Support
- ✨ Modern Glassmorphism UI
- 📱 Responsive Design
- 🎨 Smooth Hover & Click Animations


## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
